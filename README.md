# 🚀 LinkedIn AI Pro v2.0

**Akıllı LinkedIn Deneyimi İçin Açık Kaynaklı Asistan**

[![License](https://img.shields.io/badge/license-MIT-blue.svg)](https://github.com/AeroDLL/linkedin-bot/blob/main/LICENSE)
[![Tampermonkey](https://img.shields.io/badge/Tampermonkey-Required-orange)](https://www.tampermonkey.net/)
[![Gemini AI](https://img.shields.io/badge/Gemini-AI-green)](https://ai.google.dev/)

LinkedIn AI Pro, profesyonellerin LinkedIn üzerindeki etkileşimlerini akıllıca yönetmelerine yardımcı olan, tamamen açık kaynaklı bir kullanıcı komut dosyasıdır.

## ✨ Özellikler

- 🤖 **AI Destekli İçerik Üretimi** (Postlar, Yorumlar, Mesajlar)
- 🚀 **Akıllı Ağ Oluşturma** (Bağlantı önerileri & bağlantı kurma)
- 📊 **Profil Analizi & Optimizasyon**
- 📈 **Gerçek Zamanlı Analitik Panel**
- 🌍 **Çoklu Dil Desteği** (Türkçe/İngilizce)
- 🔒 **Güvenli & İnsan Benzeri Davranış**

## 🛠️ Gereksinimler

- [Tampermonkey](https://www.tampermonkey.net/) tarayıcı eklentisi
- [Gemini AI API Key](https://ai.google.dev/) (Ücretsiz)

## 📥 Kurulum

1. [Tampermonkey](https://www.tampermonkey.net/) eklentisini tarayıcınıza yükleyin
2. [LinkedIn AI Pro v2.0](https://github.com/AeroDLL/linkedin-bot/raw/main/linkedin-ai-pro.user.js) betiğini yükleyin
3. Ayarlar menüsünden Gemini API Key'inizi girin
4. LinkedIn'e giriş yapın ve araç kullanıma hazır!
